## University Online Quiz App

A full-stack educational platform that modernizes the quiz-taking experience for universities. This application provides a seamless interface for students to take quizzes while giving instructors powerful tools to manage assessments and track student performance.

### Key Features

- **Real-time Quiz Sessions**: Live quiz sessions with synchronized timing
- **Automated Grading**: Instant feedback and scoring for objective questions
- **Question Bank Management**: Comprehensive question database with categorization
- **Student Analytics**: Detailed performance tracking and progress reports
- **Multiple Question Types**: Support for multiple choice, true/false, and essay questions
- **Secure Authentication**: Role-based access control for students and instructors
- **Offline Support**: Take quizzes offline and sync when connected

### Technology Stack

#### Mobile Frontend
- **Flutter** - Cross-platform framework for iOS and Android
- **Dart** - Modern programming language for Flutter development
- **State Management** - Efficient state handling using Provider/Riverpod

#### Backend API
- **ASP.NET Core** - High-performance web framework for building APIs
- **C#** - Type-safe, modern programming language
- **Entity Framework Core** - ORM for database operations
- **JWT Authentication** - Secure token-based authentication

#### Database
- **SQL Server** - Enterprise-grade relational database management system
- **Stored Procedures** - Optimized database queries for complex operations
- **Database Indexing** - Efficient data retrieval and query performance

### Architecture

The application implements a clean, scalable architecture:

- **Mobile Layer**: Flutter application with responsive design for tablets and phones
- **API Layer**: RESTful ASP.NET Core Web API with proper versioning
- **Data Layer**: SQL Server database with normalized schema design
- **Security**: JWT token authentication with role-based authorization

### Features Implementation

#### For Students
- Browse available quizzes and assignments
- Take timed quizzes with automatic submission
- View results and detailed feedback
- Track progress over time
- Access quiz history and performance analytics

#### For Instructors
- Create and manage quiz questions
- Schedule quiz sessions
- Monitor student progress in real-time
- Generate comprehensive reports
- Export results for further analysis

### Impact

This platform has streamlined the assessment process for educational institutions, reducing manual grading time by 80% and providing students with immediate feedback on their performance, enhancing the learning experience.
